The Frontend application is built using [React + Vite + TypeScript] to ensure high performance and strict code safety during team collaboration.

Tech Stack : 
 ● Framework  : React          
 ● Build Tool : Vite
 ● Language   : TypeScript
 ● State Management : React Hooks (`useState`, `useEffect`)

How to Run Local :
1. Go into the frontend folder:
```bash
   cd frontend
   ```

Install the project dependencies:
```Bash
   npm install
```

Run the local development server:
```Bash
   npm run dev
```
